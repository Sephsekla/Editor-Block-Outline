Add outline around Gutenberg blocks while editing

== Description ==

This is a simple Gutenberg plugin that adds an outline around each block. Outline settings are based on user meta. Each user can
change the settings according to their needs.

Users can change:

-   When to show outline. On hover or always.
-   Disable outline completley.
-   Change outline color.
-   Change outline style to solid, dashed or dotted line
-   Show/hide block name
